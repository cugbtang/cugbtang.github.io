## mysql

- 为什么 MySQL 使用 B+ 树
- 为什么 MySQL 的自增主键不单调也不连续
- 为什么 MySQL 需要事务
- 为什么 MySQL 有 InnoDB 和 MyISAM 两种引擎
- 为什么 MySQL 需要索引
- 为什么 MySQL 有慢查询
- 为什么 MySQL 需要连接池
- 为什么 MySQL 有主从复制
- 为什么 MySQL 有分库分表
- 为什么 MySQL 需要 MVCC
- 为什么 MySQL 有锁机制
- 为什么 MySQL 有redo log和undo log
- 为什么 MySQL 有binlog
- 为什么 MySQL 有buffer pool
