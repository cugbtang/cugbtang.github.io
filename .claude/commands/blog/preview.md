# 预览博客文章

## 功能说明
使用此命令预览博客文章在Hugo网站上的实际效果，检查格式、布局和内容显示。

## 使用步骤

### 1. 检查Hugo服务器状态
首先检查本地Hugo开发服务器是否正在运行：
```bash
# 检查服务器状态
检查是否有hugo server进程运行

# 如果未运行，启动服务器
hugo server --buildDrafts --port 1313 &
```

### 2. 确定文章URL
根据文章的文件路径确定其在网站上的URL：
- 文件路径: `content/post/category/filename.md`
- 对应URL: `http://localhost:1313/post/category/filename/`

### 3. 检查文章内容
预览前检查文章内容：
- Front matter格式是否正确
- Markdown语法是否规范
- 代码示例格式是否正确
- 图片链接是否有效

### 4. 提供预览链接
生成文章预览链接并提供访问方式：
```
文章预览链接: http://localhost:1313/post/category/filename/
```

### 5. 检查显示效果
指导用户检查以下方面：
1. **布局检查**:
   - 标题显示
   - 作者信息
   - 日期显示
   - 标签和分类

2. **内容格式**:
   - 段落间距
   - 标题层次
   - 列表格式
   - 引用样式

3. **代码显示**:
   - 语法高亮
   - 代码块格式
   - 行号显示
   - 复制功能

4. **媒体内容**:
   - 图片显示
   - 响应式布局
   - 点击放大功能

5. **导航功能**:
   - 目录生成
   - 文章导航
   - 相关文章

### 6. 提供优化建议
根据预览结果提供优化建议：
- 格式调整建议
- 内容优化建议
- 性能优化建议
- SEO优化建议

## 常见问题解决

### 问题1: Hugo服务器未运行
**解决方法**:
```bash
# 启动Hugo服务器
hugo server --buildDrafts --port 1313 &

# 检查服务器状态
curl -I http://localhost:1313/
```

### 问题2: 文章未显示
**可能原因**:
1. draft状态为true
2. 文件路径错误
3. 构建缓存问题

**解决方法**:
```bash
# 检查draft状态
检查front matter中的draft字段

# 清理缓存
hugo --cleanDestinationDir

# 重新构建
hugo server --buildDrafts --disableFastRender
```

### 问题3: 格式显示异常
**可能原因**:
1. Markdown语法错误
2. CSS样式冲突
3. 主题兼容问题

**解决方法**:
1. 检查Markdown语法
2. 验证主题配置
3. 检查自定义CSS

### 问题4: 代码高亮异常
**解决方法**:
```toml
# 检查hugo.toml配置
pygmentsStyle = "monokai"
pygmentsCodeFences = true
pygmentsOptions = "linenos=table"
```
